Privacy Policy

August 01, 2018

This page informs you of our policies regarding the use of personal data when you use "Finder".

Information Collection And Use

Application needs different types of information for work.

Required permissions:

  • ACCESS_FINE_LOCATION and ACCESS_COARSE_LOCATION
  • RECEIVE_SMS
  • READ_SMS
  • SEND_SMS
  • INTERNET
  • ACCESS_NETWORK_STATE
  • CHANGE_WIFI_STATE
  • ACCESS_WIFI_STATE

Use of Data

"Finder" uses this permissions for various purposes:

  • LOCATION - for sending location of mobile phone and displaying your location on built-in map
  • RECEIVE_SMS - for detecting SMS-commands
  • SEND_SMS - for answering to incoming commands and sending requests
  • READ_SMS - for ability to open inbox SMS with data in case of not working broadcast receiver
  • INTERNET - for use the "Google geolocation API" (when wifi and network data received) and built-in Google map
  • ACCESS_NETWORK_STATE - network state is checked before wifi-request will be sent, for warning
  • CHANGE_WIFI_STATE - for enabling wifi to scan wifi-nets (during wifi-request)
  • ACCESS_WIFI_STATE - for determining wifi state before scanning, to leave it in the same condition

Transfer Of Data

The application does not send any information to the developers. The collected data may be sent only to the phone numbers you specify, via SMS.
